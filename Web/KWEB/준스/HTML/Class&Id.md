# Class & Id attribute 

태그 만으로 요소를 구분하거나 분류하는 것은 불가능하다 ! 

그래서 class, id 속성을 통해 특정 요소들을 분리하고 지정하여 CSS, JS와 연동한다. 

## Class Attribute 

요소를 classify할 때 사용 

이 속성 값을 class name이라고 한다. 

한 요소는 여러 class를 가질 수 있다. 

## Id Attribute 


요소 하나를 identify 

한 요소는 여러 id를 가질 수 없고, 한 id는 여러 요소에개 배정될 수 없다. 

## Naming Convention

* 대문자를 사용하거나 숫자로 시작하지 않는다.
* 단어는 하이픈(-)으로 연결한다.
